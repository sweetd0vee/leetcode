# Решения LeetCode

Репозиторий с решениями задач LeetCode на Python для подготовки к интервью.

## Структура
- `N. Problem Name.py` — решение задачи с номером `N`.
- Один файл = одна задача.

## Как использовать
- Откройте файл нужной задачи и изучите решение.
- Если в файле есть блок `if __name__ == "__main__":`, можно запустить так:
  `python "1. TwoSum.py"`
- Если блока нет, используйте класс или функцию из файла в своих тестах.

## Принципы
- Читаемый код и понятные названия
- Минимум зависимостей
- Оценка сложности рядом с решением, если уместно
